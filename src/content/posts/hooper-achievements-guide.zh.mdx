这套 **hooper basketball game** 的成就页已经上线。如果你本来就在 Build a Hooper 刷赛季，现在终于有一块板子能记住戒指、奖项、奇怪 build，以及大家爱拿来吹的纪录 —— 而不只是一张 My Card 截图。

下面讲清解锁了什么、登录后进度怎么留下。成就就是你已经在玩的同一套循环上的长期榜单，挂在这款 **hooper basketball game** 之上。

## 成就在记什么

在 **hooper basketball game** 里打开 [/achievements](/achievements)，一共 **52** 个目标，顶栏叫 **Legacy Builder**。**hooper basketball game** 的目录分成八类：

- **模式** — 分别打完 Classic / Blind / Chaos，再在各模式夺冠
- **竞技头衔** — Chaos + Blind 戒指里程碑（5、11、25、50）
- **位置** — 以 PG、SG、SF、PF、C 夺冠；五位置都齐才是 Complete Hooper
- **新秀** — 堆新秀、带新秀夺冠，或全新秀 build
- **黑马** — Chaos/Blind 且总评 ≤ A-，夺冠或不夺冠
- **Build** — 狙击手、防守墙、GOAT 蓝图一类属性形状
- **奖项** — Scoring Title、MVP、DPOY、Finals MVP，以及同局组合
- **Legacy** — 三双、60/70/73 胜、布鲁克林、无冠球队首冠、附加赛奇迹、总决赛 3-1 逆转

稀有度从普通（10 分）经 Special（200 分）到 **GOAT**（300 分）。满板 **4200** 分 —— 把 **hooper basketball game** 当生涯刷，而不是一晚通关。

## hooper basketball game 里成就怎么解锁

局内不会弹提示。赛季还是在 **My Card** 结束。登录后提交 run，打开成就页时按生涯重算 —— **hooper basketball game** 的成就不实时弹窗：

1. 在 [/game](/game) 开一局 **hooper basketball game**
2. 打完并登录提交
3. 打开 [/achievements](/achievements)

游客也能在 **hooper basketball game** 里逛完整目录。进度条和解锁日期要登录后才有 —— 跟 Build a Hooper 对 Legacy 的处理一样：先玩，想存再登。

好几条戒指梯子只认 **Chaos 和 Blind**。Classic 仍能补模式、位置、build 徽章；Dynasty、Complete Hooper 更吃难模式。只刷 Classic，板上会空掉一大块 —— 尤其是 Chaos/Blind 戒指梯、Complete Hooper、黑马和 Future GOATs。这是 **hooper basketball game** 的设计，不是漏了。

## 模式、戒指、位置

模式决定 **hooper basketball game** 在 draft 时你能看见什么。Classic：自选位置、看得见评分、三次重 roll。Blind：位置先亮、评分藏着。Chaos：位置和评分都藏到最后。这三套是 **hooper basketball game** 的骨架，模式徽章都挂在上面。

首次打完各模式最容易。模式戒指高一档。竞技头衔按 Chaos/Blind 冠军数往 Ring Collector、Eleven Rings、Dynasty Builder、Immortal Legacy 堆 —— 深刷 **hooper basketball game** 时的主线。

位置戒指任意模式都行。Complete Hooper 必须 Chaos 或 Blind 五个位置各一冠。这规则让一套 build 没法在 **hooper basketball game** 里清空整板。

## Build、新秀、黑马

Build 成就看你锁的 13 项属性。**hooper basketball game** 里常见目标：

- Sniper Build — 3PT、MID 都 ≥ A
- Floor General — PAS、HAN 都 ≥ A
- Defensive Wall — PDEF / IDEF / BLK / REB 都 ≥ B+
- Perfect Balance — 每项 ≥ A
- GOAT Blueprint — 总评 ≥ A+（97+）

新秀徽章数的是「各来源阵容 overall 最低三人」落到槽位里的数量。Trust the Kid 是入门。All-Rookie Build 要整表都是新秀。Future GOATs 要 Chaos/Blind + 全新秀 + 夺冠 —— **hooper basketball game** 板上比较响的一类。

Underdog Run / Underdog Miracle 挂在 Chaos 或 Blind、总评 ≤ A-。这种赢法跟 A+ 堆叠不是同一种爽感，也是 **hooper basketball game** 故意留的另一条路。

## 奖项与 Legacy 纪录

赛季奖项对应 Scoring King、Season MVP、Defensive Anchor、Finals MVP。同局组合解锁 Complete Superstar、Offensive Takeover、Two-Way Monster、Perfect Resume。奖项线和戒指线在 **hooper basketball game** 里并排。

Legacy 纪录来自 **hooper basketball game** 的赛季形态：

- Triple-Double Machine（单季 1 次）→ Walking Stat Sheet（生涯累计 5 次）→ Triple-Double King（单季 10+）
- 60 胜 → 70 胜 → **73-9 Club**（GOAT）
- Brooklyn Guy — 带 BKN 夺冠
- First Banner — 带从未夺冠球队夺冠
- Play-In Miracle — 附加赛进季后赛再一路夺冠
- **3-1 Finals Comeback** — 总决赛先丢三场再逆转夺冠（也是 GOAT）

这几枚 GOAT，就是 **hooper basketball game** 在拿第一冠之后还留着天花板的原因 —— 也是简单徽章刷完之后还有人回来的理由。

## 页面怎么用

[/achievements](/achievements) 是 **hooper basketball game** 的清单层。可按锁定/解锁、分类、稀有度筛选，也能搜名字。卡片显示积分、需要计数的进度，以及首次解锁日期。

成就积分和 [/leaderboard](/leaderboard) 的 Legacy 不是同一套，也不会互相累加：一边是徽章板，一边是生涯排名。两边都吃已保存的 **hooper basketball game** run。

## 现在开始可以怎么刷

在 **hooper basketball game** 里一条比较稳的路径：

1. 登录，先清 First Classic / Blind / Chaos
2. 用 Classic 试 build（Sniper、Wall、Clutch Gene）
3. 再进 Chaos/Blind 冲戒指、Complete Hooper、黑马或 Future GOATs
4. 盯奖项和胜场 —— 73 胜和总决赛大逆转本来就稀

一句话：成就把每一局打完的赛季在 **hooper basketball game** 里变成永久印章。不是每枚都公平。有意思的那些，几乎都不公平。
